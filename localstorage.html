LocalStorage Extension

USE THE EXTENSION

WHAT DOES IT DO?

Imagine local storage as variables, except unlike variables, they don't get erased when the page is reloaded. That means that instead of wasting time and effiency making a system where everyone's data gets stored on the cloud, you can use local storage so that it stays on the device, but is still there when you come back.
The blocks act just like a regular variable, except, due to limitations with the Scratch Extensions system, you have to write in the name instead of selecting it in a dropdown. Other than that, it functions identically to a scratch variable, except that it stays around when you reload the page.
